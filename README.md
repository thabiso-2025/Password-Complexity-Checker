# Password-Complexity-Checker
This is a tool that assesses the strength of a password based on criteria such as the length, presence of uppercase and lowercase letters, numbers and special characters.   

The programme ask a user for an input, this is where the user will enter their password. After the password was enter the programme will use Regular Expressions (Regex) to search for patterns within a string effciently.

The programme logic use a scoring system that use score the password on a catagories such as:
-The length of the password (>=8 characters)
-Contains a uppercase letter
-Contains a lowecase letter
-Contains special characters (e.g., !@#$%^&*)
-Contains any numbers

After the score is collect it is added up then with the sum of the points for the password a verdict is create that deeds if the password is strong (ideal password strength), modorate, and weak (which is not ideal).
